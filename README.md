# term-config-tui

TUI en Python para administrar el setup de terminal: Alacritty (colores) y Zellij (sesiones, layouts, temas).

Ver [`PLAN.md`](PLAN.md) para el plan completo.

## Estado

En desarrollo. Fase 0 / 0.5.

## Desarrollo

```bash
uv venv
uv pip install -e ".[dev]"
uv run term-config-tui
uv run pytest
```

## Instalacion (cuando este lista)

```bash
uv tool install .
# o
pipx install .
```
